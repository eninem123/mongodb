聚合与管道

聚合 aggregate 概念

  • 聚合(aggregate)是基于数据处理的聚合管道，每个文档通过一个由多个阶段（stage）组成的管道，可以对每个阶段的管道进行分组、过滤等功能，然后经过一系列的处理，输出相应的结果。
  • 语法: db.集合名称.aggregate({管道:{表达式}})

常⽤管道

  • 在mongodb中，⽂档处理完毕后， 通过管道进⾏下⼀次处理
  • 常用管道如下：
    • $group： 将集合中的⽂档分组， 可⽤于统计结果
    • $match： 过滤数据， 只输出符合条件的⽂档
    • $project： 修改输⼊⽂档的结构， 如重命名、 增加、 删除字段、 创建计算结果
    • $sort： 将输⼊⽂档排序后输出
    • $limit： 限制聚合管道返回的⽂档数
    • $skip： 跳过指定数量的⽂档， 并返回余下的⽂档
    • $unwind： 将数组类型的字段进⾏拆分

常用表达式(类似关系数据中的聚合函数)

  • 处理输⼊⽂档并输出
  • 语法：表达式:'$列名'
  • 常⽤表达式:
    • $sum： 计算总和， $sum:1 表示以⼀倍计数
    • $avg： 计算平均值
    • $min： 获取最⼩值
    • $max： 获取最⼤值
    • $push： 在结果⽂档中插⼊值到⼀个数组中
    • $first： 根据资源⽂档的排序获取第⼀个⽂档数据
    • $last： 根据资源⽂档的排序获取最后⼀个⽂档数据

格式:

    db.集合.aggregate(
      {管道1: {}},
      {管道2: {}}
    )

聚合之$group

  • 将集合中的文档分组, 可用于统计结果
  • 格式
   db.集合.aggregate(
      {$group: {_id:'$分组字段名', 显示字段:{$统计函数: '$统计字段'}}},
    )

  • _id 表示分组的依据, 使用某字段的格式:'$字段'

  • 例: 统计男生,女生各是多少

      db.stu.aggregate(
    {$group: {_id:'$gender',count:{$sum:1}}}
  )

    group文档

  • 将整个文档为一组, 指定 id 为 null

  • 求学生的总人数和平均年龄
  db.stu.aggregate(
    {$group:{_id:null,total:{$sum:1}, avg_age:{$avg:'$age'} }}
  )

透视数据: $push统计函数

$push: 把不同内容,放入一个数组(列表)

  • 统计不同性别的学生姓名

    db.stu.aggregate(
  {$group:{_id:'$gender',name:{$push: '$name'} }}
)

  • 使用: $$ROOT把这个文档内容加入到结果集中

  db.stu.aggregate(
    {$group:{_id:'$gender',name:{$push: '$$ROOT'} }}
  )

  • 动手练习

  • 数据: 

    db.t2.insert(
  [{ "country" : "china", "province" : "sh", "userid" : "a" },  
  {  "country" : "china", "province" : "sh", "userid" : "b" },  
  {  "country" : "china", "province" : "sh", "userid" : "a" },  
  {  "country" : "china", "province" : "sh", "userid" : "c" }, 
  {  "country" : "china", "province" : "bj", "userid" : "da" },  
  {  "country" : "china", "province" : "bj", "userid" : "fa" }]  
)
    • 需求：统计出每个country/province下的userid的数量（同一个userid只统计一次）
      db.t2.aggregate(
    {$group:{_id:{country:'$country', province:'$province',userid:'$userid'}}},
    {$group:{_id:{country:"$_id.country", province:'$_id.province'}, count:{$sum: 1}}}
  )

过滤数据: $match

  • 用于过滤数据,只输出符合条件的才作为输出
  • $match可以把过滤后的文档,交给下一个管道,而find()不可以
  • 过滤字段字段直接写, 不能使用 '$字段名'的形式
  • 例1: 查询年龄大于20的学生
  db.stu.aggregate(
    {$match:{age:{$gt:20}}}
  )
  • 查询年龄大于20的男生与女生人数
  db.stu.aggregate(
    {$match:{age:{$gt:20}}},
    {$group:{_id:'$gender',counter:{$sum:1}}}
  )

修改文档结构 $project

  • 作用: 修改输入文档结构, 添加字段,删除字段,重命名字段
  • 例1: 查询 只显示学生的姓名,年龄
  db.stu.aggregate(
    {$project:{_id:0, name:1, age: 1}}
  )
  • 例2: 查询男生,女生人数, 只输出性别和人数
  db.stu.aggregate(
    {$group:{_id:'$gender', count:{$sum:1}}},
    {$project:{_id:0, gender:'$_id', count:1}}
  )

动手练习

  • 数据(与上一次动手练习一样): 

    db.t2.insert(
  [{ "country" : "china", "province" : "sh", "userid" : "a" },  
  {  "country" : "china", "province" : "sh", "userid" : "b" },  
  {  "country" : "china", "province" : "sh", "userid" : "a" },  
  {  "country" : "china", "province" : "sh", "userid" : "c" }, 
  {  "country" : "china", "province" : "bj", "userid" : "da" },  
  {  "country" : "china", "province" : "bj", "userid" : "fa" }]  
)

  • 需求:

    • 统计出每个country/province下的userid的数量（同一个userid只统计一次）
    • 结果中的字段为{country:"**"，province:"**"，counter:"*"}
  db.t2.aggregate(
    {$group:{_id:{country:'$country', province:'$province',userid:'$userid'}}},
    {$group:{_id:{country:"$_id.country", province:'$_id.province'}, count:{$sum: 1}}},
    {$project:{_id:0, country:'$_id.country', province:'$_id.province', count:1}}
  )

聚合之$sort

- 将输入的文档排序后输出

  • 例1: 查询学生信息, 按年龄升序排列

    db.stu.aggregate(
  {$sort:{age:1}}
)

  • 例2: 查询男生和女生人数, 按人数降序排

    db.stu.aggregate(
  {$group:{_id:'$gender', count:{$sum:1}}},
  {$sort:{count:-1}}
)

聚合: $skip 与 $limit

  • $skip: 跳过指定数量的的文档,并返回剩余文档
    • 例1: 查询从第3条开始的学生信息
      db.stu.aggregate({$skip:2})

  • $limit: 限制聚合管道返回的文档数量

    • 例2: 查询2条学生信息
    db.stu.aggregate({$limit:2})
    • 例3: 统计男生,女生人数, 按人数升序, 取出第二条数据
    db.stu.aggregate(
  {$group:{_id:'$gender', count:{$sum:1}}},
  {$sort:{count:1}},
  {$skip:1},
  {$limit:1}
)

聚合: $unwind

  • 概念: 将⽂档中的某⼀个数组类型字段拆分成多条， 每条包含数组中的⼀个值
  • 语法: 语法：db.集合名称.aggregate({$unwind:'$字段名称'})

  • 举例:

    • 数据:
        db.t3.insert({_id:1,item:'t-shirt',size:['S','M','L']})

    • $unwind 将集合拆分

        db.t3.aggregate({$unwind:'$size'})

    • 结果如下:

      { "_id" : 1, "item" : "t-shirt", "size" : "S" }
  { "_id" : 1, "item" : "t-shirt", "size" : "M" }
  { "_id" : 1, "item" : "t-shirt", "size" : "L" }

  • 练习: 数据库中有一条数据：{"username":"Alex","tags": ['C#','Java','C++']}，如何获取该tag列表的长度？

      db.t4.insert({"username":"Alex","tags": ['C#','Java','C++']})

  db.t4.aggregate(
     {$unwind:'$tags'},
     {$group: {_id:'$username', count:{$sum:1}}},
     {$project:{_id:0,username:'$_id', count:1}}
  )

  • 在使用$unwind的时候, 默认会丢弃属性值为空的文档.

  • 如果希望保留,需要把设置 preserveNullAndEmptyArrays 为true
  • 格式为:
  db.集合名.aggregate([
    {$unwind:{
       path:'$字段名称'
       preserveNullAndEmptyArrays:true(保留)/false(丢弃)
    }
  ])
  • 验证:

  • 准备数据

    db.t5.insert([
  { "_id" : 1, "item" : "a", "size" : [ "S", "M", "L" ] },
  { "_id" : 2, "item" : "b", "size" : [ ] },
  { "_id" : 3, "item" : "c", "size" : "M" },
  { "_id" : 4, "item" : "d" },
  { "_id" : 5, "item" : "e", "size" : null }
])

  • 查询

  db.t5.aggregate({$unwind:'$size'})
  • 结果
{ "_id" : 1, "item" : "a", "size" : "S" }
{ "_id" : 1, "item" : "a", "size" : "M" }
{ "_id" : 1, "item" : "a", "size" : "L" }
{ "_id" : 3, "item" : "c", "size" : "M" }
  • 说明: 2,4,5 条数据都没有了
  • 要显示,需要保护值为null或空数组,写法如下
  db.t5.aggregate({
    $unwind:{
      path:'$size',
      preserveNullAndEmptyArrays:true
      }
    })
  • 总结
    • 管道中设置都是通过字典(对象)进行操作的.
    • 下一个管道使用上一个管道的数据时候, 需要使用 '$字段名'
